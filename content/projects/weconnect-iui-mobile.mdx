# WeConnect IUI - Réseau Social Éducatif

## Contexte du Projet

Développement d'une application mobile sociale dédiée à la communauté éducative de l'UCAC-ICAM Institute. Ce projet vise à créer un écosystème numérique pour connecter étudiants, alumni et professeurs dans un environnement collaboratif et enrichissant.

## Défis Techniques

- **Architecture Scalable**: Architecture MVC avec state management robuste
- **Multilingue**: Support natif anglais/français avec ARB files
- **Temps Réel**: Système de chat intégré avec Supabase
- **UX/UI**: Design Material adapté aux besoins académiques
- **Performance**: Optimisation pour devices low-end

## Solutions Implémentées

### Architecture Mobile

- **Flutter 3.19** avec Dart 3.4.1 pour la cross-platform
- **Architecture MVC** avec séparation claire des responsabilités
- **Provider** pour la gestion d'état réactive
- **Supabase** pour l'authentification et base de données temps réel

### Fonctionnalités Clés

- Système d'authentification sécurisé (Login/Signup)
- Profils utilisateurs personnalisables
- Chat en temps réel entre membres
- Onboarding intuitif pour nouveaux utilisateurs
- Système de paramètres avancé

### Internationalisation

- Support multilingue avec fichiers ARB
- Interface adaptive selon la locale
- Extensibilité pour nouvelles langues

## Technologies Utilisées

- **Mobile**: Flutter 3.19.0, Dart 3.4.1
- **Backend**: Supabase (Auth, Firestore, Storage)
- **State**: Provider pattern
- **UI/UX**: Material Design 3
- **I18n**: Flutter Intl avec ARB files
