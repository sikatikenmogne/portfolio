# Green market - Plateforme E-commerce

## Contexte du Projet

Développement d'une plateforme e-commerce complète dédiée à la vente de produits agricoles biologiques. Le projet vise à connecter producteurs locaux et consommateurs dans un écosystème numérique moderne, robuste et scalable.

## Défis Techniques

- **Architecture Full Stack**: API REST robuste avec interface utilisateur réactive
- **Performance**: Optimisation pour gestion de catalogue large et trafic élevé
- **Sécurité**: Authentification sécurisée et gestion des paiements
- **Scalabilité**: Architecture containerisée pour déploiement cloud
- **UX Commerce**: Interface intuitive pour expérience d'achat optimale

## Solutions Implémentées

### Architecture Backend

- **Django 4.2** avec Django REST Framework pour API robuste
- **PostgreSQL** pour gestion des données complexes
- **Authentication JWT** et gestion granulaire des permissions
- **Docker** pour containerisation et déploiement simplifié

### Frontend Moderne

- **Vue.js 3** avec composition API pour réactivité optimale
- **Tailwind CSS** pour design system cohérent et responsive
- **pnpm** pour gestion performante des dépendances
- **Architecture modulaire** avec composants réutilisables

### Fonctionnalités E-commerce

- Catalogue produits avec recherche/filtrage avancé
- Système de panier et gestion des commandes
- Profils utilisateurs et historique d'achats
- Système de reviews et ratings produits
- Interface d'administration complète

## Technologies Utilisées

- **Backend**: Django 4.2.5, DRF 3.14.0, PostgreSQL 13.7
- **Frontend**: Vue.js 3.2.47, Tailwind CSS 3.3.2
- **DevOps**: Docker 24.0.5, GitHub Actions
- **Tools**: Poetry, pnpm 9.6.0, Swagger/OpenAPI
- **Testing**: Django Test Framework, Jest
